# Desarrollo de aplicaciones iOS

Repositorio correspondiente a la materia **Desarrollo de aplicaciones iOS**.  
En este proyecto se incluyen diferentes ejercicios y aplicaciones desarrolladas utilizando **Swift** y **UIKit** en **Xcode**, con el objetivo de practicar conceptos fundamentales del desarrollo de aplicaciones móviles para el ecosistema de Apple.

## Contenido del repositorio

Este repositorio contiene dos aplicaciones de ejemplo:

### 1. Contactos

Aplicación sencilla que permite gestionar una lista de contactos.  
Su objetivo principal es practicar la creación de interfaces, navegación entre pantallas y manejo básico de datos en una aplicación iOS.

**Características principales:**

- Visualización de una lista de contactos.
- Navegación entre pantallas utilizando controladores de navegación.
- Interfaz construida con UIKit.
- Ejemplo de estructura básica de una aplicación iOS.

### 2. Recordatorios

Aplicación simple para gestionar recordatorios o tareas pendientes.

**Características principales:**

- Visualización de una lista de recordatorios.
- Agregar nuevos recordatorios.
- Actualización dinámica de la interfaz cuando cambia la información.
- Ejemplo práctico del manejo de datos dentro de la aplicación.

## Tecnologías utilizadas

- **Swift**
- **UIKit**
- **Xcode**
- **iOS SDK**

## Requisitos

Para ejecutar las aplicaciones es necesario contar con:

- **macOS**
- **Xcode** (versión reciente)
- **iOS Simulator** o un dispositivo iOS físico

## Objetivos del repositorio

El objetivo de este repositorio es servir como material de práctica para comprender los fundamentos del desarrollo de aplicaciones iOS, incluyendo:

- Estructura de una aplicación en Swift.
- Manejo de controladores de vista.
- Navegación entre pantallas.
- Gestión básica de datos en aplicaciones móviles.
